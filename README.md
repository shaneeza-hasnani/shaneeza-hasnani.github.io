# shaneeza-hasnani.github.io

Personal portfolio website for Shaneeza Hasnani, CFE.

## Setup

1. Clone the repository
2. Add your `resume.pdf` file to the root directory (required for the Resume navigation link)
3. Open `index.html` in a browser or deploy to GitHub Pages

## Features

- Responsive design
- Interactive animations
- Skills showcase
- Work experience timeline
- Projects gallery
- Education and certifications
- Contact information with social links
